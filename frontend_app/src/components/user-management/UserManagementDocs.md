# User Management Guide

## Overview
This page describes how to use the User Management features for admin users.

## Features
- List all users
- Filter and search by email or role
- Change user roles (admin/standard)
- Reset user passwords
- Delete users (single or bulk)
- Bulk role changes
- Accessible and robust error handling

## How to Access
- Only users with the `admin` role see the "User Management" menu item in the sidebar.
- Navigate to `/user-management` to access the admin UI.

## Usage
- Use the filter and role dropdowns to search users.
- Select users with the checkbox for bulk actions.
- Use the action buttons to update roles, reset passwords, or delete users.

## Troubleshooting
- If you see errors, check your network connection and authentication status.
- Only admin users can access these features.
- For migration or advanced operations, contact your system administrator.
