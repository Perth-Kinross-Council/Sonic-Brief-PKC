# User Management Component

## Overview
This directory contains the user management UI and logic for admin users, including:
- User listing, filtering, and searching
- Role management (admin/standard)
- User deletion
- Bulk actions
- Error boundaries and accessibility

## Usage
- Only users with the `admin` role can access these features.


## Testing
- Run `npm test` or `pnpm test` to execute tests in this directory.
